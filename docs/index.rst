Project Overview
==============

A Python application for rocket flight simulation using OpenRocket, with wind data integration and Monte Carlo analysis.

Components
=========

GUI Interface
-----------
The GUI module handles all user interactions and input collection.

Data Formatter
------------
The Data Formatter processes and prepares wind data for simulation use.

Simulation Manager
---------------
The Simulation Manager handles OpenRocket integration and runs the simulations.
